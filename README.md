# #500MusicalIdeasChallenge

This repository is a dedicated laboratory for my journey through 500 musical ideas using the **Strudel REPL**. 

## 🎯 The Goal
To build consistency and simplicity by pushing one musical sketch regularly. These are ideas actively being studied and the files might evolve with the idea.

## 🛠️ How to use
You can copy the code from any `.js` file in this repo and paste it directly into [strudel.cc](https://strudel.cc) to hear the idea.
